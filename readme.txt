1.环境：本地需要nodejs环境 如无node环境需要自行下载
2.代码逻辑：全局搜索【二开标识】可查看相关逻辑函数作用
3.本地开发：运行项目需安装依赖 命令行执行 【npm i 】因只开发了web版本 所以进入【web-》flat-web】在执行一次 【npm i 】 执行成功后 执行【npm run start】开始本地调试/yarn start
4.打包：按照步骤1后 命令行执行 【npm run build】
5.部署：按照步骤3后 把【dist】文件夹所有文件上传到ftp服务器 完成前端项目部署
6.tips因为dist文件夹所有文件带hash值 为减少ftp过期文件占用资源 可事先删除ftp文件夹所有前端文件再上传
